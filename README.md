# Kiran Bandaru – Portfolio

A professional, modern personal portfolio built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## Features

- Dark mode, minimalist design with Cobalt Blue accent
- Sticky navbar with smooth scroll to sections
- Hero, About, Experience (vertical timeline), Skills grid, Contact
- Responsive layout for mobile and desktop
- Fade-in and scroll-triggered animations

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Profile image

Place your profile photo at `public/profile.jpg` or `public/profile.png.jpg` so it appears in the hero section. Use a square or portrait image for best results.

## Resume

Place your resume PDF at `public/resume.pdf` so the “Download Resume” button works. If the file is missing, the link will still point to `/resume.pdf` (you can replace it later).

## Build

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
